1. Node.js?
=> 크롬의 v8엔진으로 빌드된 자바스크립트 런타임 환경이다.


2. REPL이란?
REPL(Read Eval Print Loop: 입력 수행 출력 반복)
=> 터미널에서 Node.js 환경을 구동시켜 간단하게 실행해 볼 수 있다.


3. 노드 특징

3-1. 이벤트 기반
이벤트가 발생할 때 미리 지정해둔 작업을 수행하는 방식
3-2. 논 블로킹 i/o
오래 걸리는 함수를 백그라운드로 보내서 다음 코드가 먼저 실행되게 하고, 나중에 오래 걸리는 함수를 실행
파일 입출력이나 네트워크 요청등은 백그라운드에 보낸다. 그리고 다음 코드를 실행하고 오래 걸리는 코드를 실행한다.
3-3. 싱글 쓰레드
but 멀티 쓰레드를 사용할 수 있도록 worker_쓰레드 모듈이 존재


4. 노드의 장단점
장점 : I/O 작업에 효율적이다./ 웹 서버가 내장되어 있다./ 멀티 쓰레드 방식에 비해 컴퓨터 자원을 적게 사용한다.
단점 : 싱글 쓰레드이기 때문에 CPU 코어를 하나만 사용/ CPU 많은 작업에 비효율적/ 하나뿐인 쓰레드 관리 필수
