4. Rest API Authentication
=> 무상태이기 떄문에 세션을 사용하지 않는다.
=> 각 요청이 인증에 필요한 데이터가 필요하다.
=> Rest api는 서버에 클라이언트에 대한 정보를 저장하지 않는 것이 원칙이다.
=> 클라이언트에 토큰을 부여한다.
